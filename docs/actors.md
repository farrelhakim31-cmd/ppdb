# Aktor Sistem

## Daftar Aktor dan Peran

| Aktor | Deskripsi Peran |
|-------|----------------|
| Calon Siswa | Mendaftar, melengkapi data & berkas, melakukan pembayaran, memantau status. |
| Admin Panitia | Mengelola master data, memantau pendaftar, membuat laporan, mengatur akun. |
| Verifikator Administrasi | Memeriksa kelengkapan & kebenaran data/berkas pendaftar. |
| Keuangan | Membuat/mengecek tagihan, memverifikasi bukti pembayaran. |
| Kepala Sekolah/Yayasan | Melihat dashboard eksekutif & ringkasan KPI. |
| Sistem (Otomatis) | Mengirim notifikasi, mencatat audit log, pembaruan status otomatis. |
| Pengunjung/Publik (opsional) | Melihat info pendaftaran & mengakses link registrasi. |

## Detail Peran Aktor

### 1. Calon Siswa
- Registrasi akun baru
- Login ke sistem
- Mengisi formulir pendaftaran
- Upload dokumen persyaratan
- Melakukan pembayaran
- Melihat status pendaftaran
- Menerima notifikasi

### 2. Admin Panitia
- Mengelola data master (jurusan, gelombang, dll)
- Monitoring pendaftar
- Verifikasi data pendaftar
- Generate laporan
- Manajemen user account
- Konfigurasi sistem

### 3. Verifikator Administrasi
- Review dokumen pendaftar
- Verifikasi kelengkapan berkas
- Approve/reject pendaftaran
- Memberikan catatan verifikasi

### 4. Keuangan
- Membuat tagihan pembayaran
- Verifikasi bukti pembayaran
- Update status pembayaran
- Laporan keuangan

### 5. Kepala Sekolah/Yayasan
- Dashboard eksekutif
- Laporan ringkasan
- Monitoring KPI pendaftaran
- Approval kebijakan

### 6. Sistem (Otomatis)
- Kirim email/SMS notifikasi
- Generate nomor pendaftaran
- Update status otomatis
- Backup data
- Audit logging

### 7. Pengunjung/Publik
- Akses informasi pendaftaran
- Download brosur/panduan
- Registrasi awal
